# 4-Digit 7-Segment Display

This project shows how to control a 4-Digit 7-Segment display.

Author: **Huy Dang**

### Component Required

- (1) x Elegoo Uno R3- (1) x Breadboard- (1) x 74HC595- (1) x Four Digital Tube Segment Display 
- (4) x 220 ohm resistor- (many) x M-M wires
